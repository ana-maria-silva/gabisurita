# Exemplos — Vibecoding

Cada subdiretório contém uma "fotografia" do projeto em um checkpoint específico do tutorial. Use esses arquivos como referência para comparar com o que você construiu.

| Diretório | Checkpoint | O que tem |
|---|---|---|
| `checkpoint-1-hello-world/` | Parte 1.3 | `index.html` mínimo com "Hello, World!" |
| `checkpoint-2-dashboard/` | Parte 2.2 | Dashboard visual com 3 cards de métricas, tema escuro, data dinâmica |
| `checkpoint-3-graficos-e-listas/` | Parte 3.2 | Dashboard + gráfico de barras (Chart.js) + lista de 4 eventos |

> **Checkpoint 4 (Integração com Google APIs)** não tem exemplo aqui pois requer credenciais pessoais — cada pessoa vai ter a sua própria versão.

## Como usar

Abra qualquer `index.html` diretamente no navegador para ver o resultado esperado de cada etapa.

```bash
# Exemplo: abrir o checkpoint 3 no macOS
open checkpoint-3-graficos-e-listas/index.html
```
